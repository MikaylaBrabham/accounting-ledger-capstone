# 💻 Accounting Ledger CLI Application

## 📌 Overview
This project is a **Command Line Interface (CLI) Accounting Ledger Application** built in Java.

It allows users to:
- Record financial transactions
- Store them in a CSV file
- View and filter transaction history
- Generate reports

---

## 🚀 Features

### 🏠 Home Screen
- Add Deposit
- Make Payment
- View Ledger
- Exit Application

---

### 📒 Ledger Screen
- View All Transactions
- View Deposits Only
- View Payments Only
- Access Reports

---

### 📊 Reports Screen
- Month To Date
- Previous Month
- Year To Date
- Previous Year
- Search by Vendor

---
